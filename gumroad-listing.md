DroidPDF — The affordable PDF library for Android developers

DroidPDF is a Kotlin-native PDF library packaged as an Android .aar. It covers the full spectrum of PDF operations at a fraction of the cost of iText7.

ISO 32000-1 (PDF 1.7) compliant.


What's included

• PDF Generation — Text, images (JPEG/PNG/WebP), tables with a high-level layout API
• Font Support — Standard 14 fonts + TTF/OTF embedding (CJK ready via Apache FontBox)
• Page Operations — Merge, split, rotate, reorder, insert, delete pages
• 11 Annotation Types — Highlights, underlines, sticky notes, stamps, shapes, ink, hyperlinks
• AcroForm — Text fields, checkboxes, radio buttons, dropdowns
• AES Encryption — 128-bit and 256-bit with granular permission controls
• Text Extraction — Extract and search text from existing PDFs
• Lenient Parser — Reads malformed PDFs without crashing


Quick Start

val pdf = PdfDocument(outputStream)
val document = Document(pdf)
document.add(Paragraph("Hello, DroidPDF!").setFontSize(16f))
document.add(Table(3).apply {
    addCell("Name")
    addCell("Age")
    addCell("City")
})
document.close()
