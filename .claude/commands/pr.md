# PR作成

現在のブランチの変更内容からPull Requestを作成してください。

## 手順

1. `git diff main...HEAD` で全変更を確認
2. コミット履歴を確認
3. 以下のテンプレートでPRを作成

## PRテンプレート

```
## 概要
{1-3行で変更の目的}

## 変更内容
- {変更点を箇条書き}

## 対応Phase
Phase {X}: {Phase名}

## テスト
- [ ] 単体テスト追加/更新
- [ ] 既存テストが全てパス
- [ ] {機能固有のテスト項目}

## PDF仕様準拠
- [ ] ISO 32000-1 (PDF 1.7) に準拠
- [ ] 他のPDFビューア（Adobe Acrobat / Chrome / Android標準）で表示確認

## チェックリスト
- [ ] Lenient方針: 不正入力でクラッシュしない
- [ ] メモリ: ページ単位の遅延読み込みに沿っている
- [ ] スレッドセーフティ前提に反していない
```
