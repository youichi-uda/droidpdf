# PDF検証

生成したPDFファイルの妥当性を検証してください。

## 検証手順

1. **構造検証**: PDFファイルのバイナリ構造を確認
   - `%PDF-1.7` ヘッダーの存在
   - `%%EOF` トレーラーの存在
   - xref（クロスリファレンステーブル）の整合性
   - オブジェクト番号の連続性

2. **コンテンツ検証**: 指定された内容が正しく含まれているか
   - テキストストリームの存在
   - フォントリソースの埋め込み
   - 画像リソースの埋め込み
   - ページ数の一致

3. **相互運用性チェック**: 可能であれば以下を確認
   - Android PdfRenderer で開けるか
   - ファイルサイズが妥当か（異常に大きくないか）

$ARGUMENTS に検証対象のPDFファイルパスを指定してください。
